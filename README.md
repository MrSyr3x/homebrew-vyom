# Mrsyr3x Vyom

## How do I install these formulae?

`brew install mrsyr3x/vyom/<formula>`

Or `brew tap mrsyr3x/vyom` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mrsyr3x/vyom"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
